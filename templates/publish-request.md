# 博客发文指令模板

以后你可以直接按下面格式下达命令，我会按仓库规范直接落稿。

```md
帮我发表文章
栏目：业主维权
日期：2026-03-18
状态：草稿
slug：自动
标签：自动

正文：
# 物业费怎么交，才能实现双赢？

这里粘贴 Markdown 正文
```

## 字段说明
- `栏目`：必须由你指定，我不自动判断。
- `标题`：可省略。默认从正文中的一级标题提取。
- `日期`：使用 `YYYY-MM-DD`。
- `状态`：只写 `草稿` 或 `发布`。
- `slug`：写 `自动` 时，我会按仓库规则生成；你也可以直接指定。
- `标签`：写 `自动` 时，我会按 `data/tag-whitelist.toml` 选择 3–5 个标签；你也可以自己指定。
- `正文`：直接粘贴 Markdown 正文。

## 最简版本
如果你不想每次都写完整，也可以至少给我这几个字段：

```md
帮我发表文章
栏目：业主维权
日期：2026-03-18
状态：草稿

正文：
# 你的标题

你的正文
```

缺失的 `slug`、`标签` 和 `标题字段` 我会自动按规范补齐；前提是正文里有明确一级标题。
